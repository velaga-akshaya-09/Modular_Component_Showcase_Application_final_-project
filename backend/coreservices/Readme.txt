Core Services Backend

This folder contains the Spring Boot backend for the Modular Component Showcase Application.

Main features:
- JWT authentication
- Role-based access control
- Component CRUD APIs
- Category CRUD APIs
- PostgreSQL persistence

Run command:
./mvnw spring-boot:run

Windows:
mvnw.cmd spring-boot:run

If Maven is not installed globally, use:
mvn.cmd spring-boot:run
